# Hubot Nietzche

A hubot plugin deal with Hooroo posses.

## Installation

In hubot project repo, run:

```
npm install hubot-posse
```

Then add this line:

**hubot-nietzche**

to your `external-scripts.json`

## Hubot Commands
```
hubot posse me [<number>]
```

- number (optional): The number of quotes to display (max. 10)
